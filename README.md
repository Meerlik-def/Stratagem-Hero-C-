# Stratagem Hero
Fast‑paced C++ terminal arcade inspired by the Stratagem Hero mini‑game from Helldivers 2.
Created by **Meerlik-def**.
**Stratagem Console Hero** is a reaction‑based console game where you must quickly repeat randomly generated stratagem sequences under time pressure.
The game challenges your speed, precision, and focus, while rewarding consistency with increasing difficulty.

*Your ultimate mission:*
_Can you survive long enough to reach **999 points**?_

The game includes arcade‑style sound effects, dynamic sequence generation, and a clean minimalistic terminal interface.

Why One .cpp File?

This project was intentionally written as one self‑contained C++ file to make it:

    Easy to copy into any IDE

    Simple to integrate into student projects

    Fast to compile and run without setup

    Perfect for learning console input, timing, and sound

This design choice makes the game extremely portable and beginner‑friendly.

Created by **Meerlik** — crafted with love for C++, arcades, and Helldivers‑style chaos.
